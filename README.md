# 🐱 Pokémon API Viewer

A simple web application that fetches and displays Pokémon data using the PokeAPI.  
Built with **vanilla JavaScript**, focusing on API usage, DOM manipulation, and event handling.

---

## 🚀 Features

- 📡 Fetches Pokémon data from a public API
- 📋 Populates a dropdown with Pokémon names
- 🖼️ Displays Pokémon image (front & back)
- 🔄 Click image to toggle between front and back sprites
- 📊 Shows detailed Pokémon information:
  - Name
  - Height
  - Weight
  - Base experience
  - Abilities
- ⚠️ Error handling for failed API requests
- 🧼 Clean and simple UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
